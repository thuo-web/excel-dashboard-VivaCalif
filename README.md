
# VivaCalif Sales Dashboard

## Overview
The **VivaCalif Sales Dashboard** is an interactive Excel-based visualization designed to uncover customer behavior, sales trends, delivery performance, and product popularity across California.  
Using over **2,400 transactions**, the dashboard transforms raw data into a clear, decision-ready report.


## Key Metrics
Displayed on the left panel of the dashboard:

- **Total Orders:** 2,400  
- **Total Sales Amount:** \$649.0K  
- **Total Quantity Sold:** 11,897  
- **Average Customer Rating:** 4.0  
- **Average Delivery Days:** 3  

These KPIs automatically update when filters are applied.

All visuals and KPIs update instantly to reflect selected filters.
